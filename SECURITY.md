# Security Policy

## Supported Versions

Please use the latest version of PromptForge for the best security and features.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1    | :x:                |

## Reporting a Vulnerability

We take the security of PromptForge seriously. If you believe you have found a security vulnerability, please report it to us as follows:

1. **Do not open a public issue.**
2. Send an email to [INSERT EMAIL ADDRESS] with details of the vulnerability.
3. We will acknowledge your report within 48 hours and provide a timeline for a fix.

Thank you for helping keep PromptForge secure!
